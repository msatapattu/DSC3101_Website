Contingency Table

  • A common way to represent and analyze categorical data is through contingency tables.

  • Let \(X\) and \(Y\) be categorical variables with \(I\) and \(J\) levels respectively. Consider a rectangular table having \(I\) rows for categories of \(X\) and \(J\) columns for categories of \(Y\).

  • The cells of the table represent the \(IJ\) possible combinations of outcomes. When the cells contain frequencies of outcomes, the table is called a contingency table (or a cross-classification table).

Contingency Table

  • A contingency table with \(I\) rows and \(J\) columns is called an \(I\)-by-\(J\) (denoted by \(I \times J\) ) table.

Joint probability distribution

  • If subjects are randomly sampled from the population and cross-classified, both \(X\) and \(Y\) are random and \((X,Y)\) has a bivariate discrete joint distribution.

  • Let \(\pi_{ij} = P(X = i, Y = j)\), the probability of falling into the \((i, j)^{th}\) (row,column) in the table.

  • We can use the table() function to create a contingency table in R.

  • Let’s consider an example with random data, where the variable \(Y\) represents the drug type (Pethidine or Morphine) that used in a study and three pain severity levels of variable \(X\).

Proportions of the Contingency Tables

  • We can get the estimated probabilities by using the frequencies in the cells of contingency tables.

  • The function prop.table() can be used to get the sample probabilities overall as well as by rows and columns.

Proportions of the Contingency Tables

prop_tab <- prop.table(cross_tab1) # sample joint probabilities
prop_tab
          Y
X          Morphine Pethidine
  Moderate      0.2       0.2
  Severe        0.1       0.0
  Slight        0.2       0.3
  • By default, the function calculates the proportion of each cell respect to the total of observations, so the sum of the cells is equal to 1.

Proportions of the Contingency Tables

  • The margin argument in prop.table() function allows to set the index (1: rows, 2: columns) according to which the proportions are going to be calculated.

  • If we set margin = 1, the sum of each row will be equal to 1 in the resulting table.

  • If we set margin = 2, the sum of each column will be equal to 1 in the resulting table.

Proportions of the Contingency Tables

prop.table(cross_tab1, margin = 1) 
          Y
X          Morphine Pethidine
  Moderate      0.5       0.5
  Severe        1.0       0.0
  Slight        0.4       0.6
prop.table(cross_tab1, margin = 2)
          Y
X          Morphine Pethidine
  Moderate      0.4       0.4
  Severe        0.2       0.0
  Slight        0.4       0.6

Graphical Summaries of Two way Contingency Tables

  • In order to plot a contingency table in R we can use of the bar charts with the barplot() function.

  • There are two main types: grouped and stacked. By default, the function will create a stacked bar plot.

Bar charts for Contingency Tables

colors <- c("red","blue","green")
barplot(cross_tab1, col= colors)

# Adding a legend
legend("topright", legend = c("Moderate", "Severe", "Slight"), fill = colors)

Bar charts for Contingency Tables

  • If you prefer a grouped bar plot you will need to set the argument beside as TRUE.
colors <- c("red","blue","green")
barplot(cross_tab1, col= colors, beside = TRUE)

# Adding a legend
legend("topleft", legend = c("Moderate", "Severe", "Slight"), fill = colors)

Mosaic plots

  • An alternative to barplots are mosaic plots, that allows you to display two or more categorical variables.
mosaicplot(cross_tab1)
  • All the boxes across categories having the same area is a signal of independence.

Statistical Tests for Contingency Tables

  • We can apply the Chi-squared Test and Fisher’s Exact Test,
  1. To test whether the two variables in the contingency table is independent or not.
  2. To test for equality or homogeneity of proportions between two or more groups.

Chi-squared Test in R

  • Suppose that we are interested to test whether the education level and Wage category are independent or not, by using the Wage dataset.
  • Consider the hypotheses,

\(H_0:\) Education level and Wage category are independent vs

\(H_1:\) Education level and Wage category are not independent.

Chi-squared Test in R: Example

cross_tab1 # Corresponding contingency table
                    
                     Above Below
  1. < HS Grad          46   222
  2. HS Grad           302   669
  3. Some College      312   338
  4. College Grad      464   221
  5. Advanced Degree   359    67
chisq.test(cross_tab1) # Chi-square test in R

    Pearson's Chi-squared test

data:  cross_tab1
X-squared = 541.37, df = 4, p-value < 2.2e-16
  • We reject the null hypothesis at 5% level of significance and there is no enough evidence to conclude that the two variables are independent.

Fisher’s Exact Test in R

  • If at least one expected frequency of the contingency table is less than 5, the Chi-squared test is not applicable. Then, we use Fisher’s Exact Test as a small sample test.
#Fisher's Exact Test 
Test <- matrix(c(1,8,4,5), ncol=2, byrow=TRUE) # Contingency table
fisher.test(Test) #Two sided
