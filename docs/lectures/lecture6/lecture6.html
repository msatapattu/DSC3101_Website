Multivariate Probability distributions

  • Multivariate distributions are designed to describe the probability distributions of more than one random variable at the same time.

  • Since the variables are often correlated, exploring them individually would only provide limited insight about the data.

Multivariate Probability distributions

  • We will discuss about,
    • Multivariate Normal Distribution
    • Multivariate t- distribution

Structure of Multivariate data

  • Rectangular in shape - organized by rows and columns.

  • The mean identifies the location of the distribution in a multidimensional space. (Mean vector)

  • The variance-covariance measures spread of multivariate data in several directions (variance covariance matrix)

Multivariate Probability Distributions

1. Multivariate Normal Distribution

  • The multivariate normal distribution is a generalization of the univariate normal distribution to two or more variables.

  • It is a distribution for random vectors of correlated variables, where each vector element has a univariate normal distribution.

Multivariate Normal Distribution

  • In the simplest case, no correlation exists among variables, and elements of the vectors are independent univariate normal random variables.

  • Because it is easy to work with, the multivariate normal distribution is often used as a model for multivariate data.

  • The multivariate normal distribution uses the mean vector and covariance matrix as the parameters.

Multivariate Normal Distribution in R

library(MASS)

# set seed and create data vectors
set.seed(1)
                                
mu.sim <- c(0, 2)                               
sigma.sim <- matrix(c(10, 5, 2, 9),
                                ncol = 2)

# create bivariate normal distribution
sample_distribution <- mvrnorm(n = 100,
                            mu = mu.sim,
                            Sigma = sigma.sim)
