DSC 3101 - Advanced Statistical Applications II

Introduction

Advanced Statistics Applications II course teaches students to use statistical theory to solve real-life problems using Statistical tools and software packages. This course will help enhance their technical knowledge through laboratory sessions. The content of this course mainly covers the theoretical concepts such as Statistical Simulation Methods, Multivariate Data Analysis, Nonparametric Statistical Inference and Categorical Data Analysis. It also includes some practical sessions where students learn how to use more recent R packages for complex data manipulation and creating academic documents.

Expectation of the Class

  • Not to become an expert in all statistical software packages but to become an expert data scientist

  • Identify appropriate software packages available for advanced statistical data analysis techniques.

  • Apply the advanced statistical methods in practice and interpret the results.

  • Perform a comprehensive statistical analysis, and compile and self publish a scientific report.

  • Self-explore the software tools available for statistical data analysis.
